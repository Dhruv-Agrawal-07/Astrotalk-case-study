Overview:
 Astrotalk is a leading astrology and mental wellness platform offering Vedic astrology, tarot, vaastu, and spiritual guidance through paid consultations.
This case study analyzes the current app experience, identifies user pain points, and proposes UX/UI improvements to enhance user engagement and conversion.

Methodology:
 Framework Used: CIRCLES Method(Comprehend, Identify, Report, Cut, List, Evaluate, Summarize)
Tools Used:
Figma
User Surveys
Heuristic Evaluation
Competitive Analysis

Outcome:
 Proposed design improvements aim to:
Improve search clarity and trust
Enhance engagement with personalized recommendations
Simplify decision-making for varied user types
Create a more intuitive, conversion-optimized UI
